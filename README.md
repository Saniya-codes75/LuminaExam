## 🚀 Project Overview
**Lumina Exam** is a high-security, web-based platform designed to provide a seamless testing environment. Developed as part of my **BCA** curriculum, it solves the challenge of remote academic integrity through real-time monitoring.

## 🛡️ Key Technical Features
* **Anti-Cheat Logic:** Custom-built tab-switching detection and focus monitoring.
* **Automated Supervision:** Real-time exam timers with mandatory auto-submission.
* **Role-Based Access:** Secure dashboards for Administrators (Exam Management) and Students (Assessment).
* **Modern UI:** Responsive frontend built with **Bootstrap** and **Glassmorphism** design principles.
* **Result Automation:** Instant scoring and feedback generation for objective tests.

## 🛠️ Technical Stack
* **Backend:** Python / Django
* **Frontend:** JavaScript (ES6+), HTML5, CSS3, Bootstrap
* **Database:** MySQL / SQLite
* **Security:** Session tracking and monitoring logic
